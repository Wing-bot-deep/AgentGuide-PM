# 面试逐字稿生成器

> 输入你的项目经历要点，AI 按 STAR 结构生成可直接背诵的面试回答逐字稿。

## 使用说明

1. 选择逐字稿类型（自我介绍 / 项目介绍 / 离职原因 / 职业规划）
2. 填写你的基本信息和项目要点（不需要完整，关键词即可）
3. AI 生成完整逐字稿，时长控制在 2-3 分钟
4. 对不满意的部分可以修改要点后重新生成

::: tip 使用建议
生成后大声朗读一遍，超过 3 分钟的需要精简。面试回答节奏：自我介绍 ≤2 分钟，单个项目介绍 3-4 分钟。
:::

## 自我介绍框架参考

如果不知道从何写起，可以参考以下框架：

```
[1] 一句话定位（当前身份 + 核心标签）
    "我是一名有 X 年经验的产品经理，专注 AI 产品方向。"

[2] 核心经历（最相关的 1-2 段经历，各一句话）
    "在 XX 公司负责 AI 客服产品，主导了 RAG 系统从 0 到 1 的落地。"

[3] 代表性成果（数字说话）
    "该产品上线后，人工客服工作量降低 40%，用户满意度提升 17pp。"

[4] 为什么来这里（表达真实动机）
    "我对贵司在 Agent 方向的布局很感兴趣，尤其是 XX 产品的思路。"
```

## STAR 框架回答项目经历

| 部分 | 说什么 | 时长 |
|------|-------|------|
| S - Situation | 项目背景、用户问题、为什么要做 | 20秒 |
| T - Task | 你的角色、目标是什么 | 15秒 |
| A - Action | 你做了什么关键决策，遇到什么挑战怎么解的 | 2分钟 |
| R - Result | 核心数据结果 + 你的成长 | 30秒 |
