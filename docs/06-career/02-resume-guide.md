## 简历核心结构

```
[个人信息]    姓名 / 联系方式 / GitHub（可选）

[一句话定位]  「X 年 AI PM 经验，专注 [细分方向]，曾主导 [代表项目]」

[工作经历]    每段经历 3-4 条，STAR 结构，数字量化
              ✅ 「主导 RAG 知识库产品从 0 到 1，DAU 1.2 万，满意度 89%」

[项目经历]    独立项目 / 副业项目，突出 AI 技术应用

[技能]        技术：RAG / Agent / Fine-tuning / Prompt Engineering
              工具：Figma / SQL / Python（基础）
              证书：可选
```
