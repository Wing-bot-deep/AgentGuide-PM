## AI PM 作品集应该展示什么

1. **AI 产品案例**：设计思路 + 技术选型理由 + 数据结果
2. **Prompt 工程实践**：你写的 System Prompt，附效果对比
3. **技术理解文章**：用产品语言解释 RAG / Agent 等技术
4. **Vibe Coding 项目**：用 AI 工具搭建的小产品，展示执行力

## 常见误区

- ❌ 把 PPT 方案当作品集（没有落地数据）
- ❌ 作品集里没有 AI 相关内容（和普通 PM 没有区别）
- ❌ 过度包装，夸大个人贡献（面试时无法自圆其说）
