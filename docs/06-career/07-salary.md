# 谈薪实战指南

> 本内容整理自真实招聘经验，适用于校招生和社招转型的 AI PM。

---

## 核心原则：谈薪的 3 个关键点

根据 HR 的实际经验，遇到以下三类候选人，通常会建议业务部门给到预算内最高薪资：

1. **不着急报数，先问清公司薪资结构**
2. **能清晰给出具体期望薪资，而非模糊表述**
3. **薪资超预算时，会合理争取且逻辑清晰**

---

## 第一步：先问清薪资结构，不要急着报价

### 为什么要这样做？

主动询问薪资结构，说明你对求职态度认真、做过功课，HR 会认为这类候选人更值得重点评估。

### 标准提问话术

```
"面试官您好，因为我对贵公司的薪资结构不太了解，
能否简单介绍一下这个岗位的薪资构成？
比如固定部分、绩效部分是如何设置的？"
```

### 进阶追问（根据节奏逐步问）

- "咱们薪资里，绩效占比多少？考核标准是什么？"
- "公司一年有几次调薪机会？调薪比例大概是多少？"
- "除了工资，还有哪些补贴或福利？"
- "试用期薪资是转正薪资的多少比例？"

---

## 第二步：清晰给出具体期望薪资

### ❌ 千万不要这样说

| 错误说法 | HR 内心想法 | 后果 |
|---------|-----------|------|
| "期望薪资 1.5-2 万" | 既然最低能接受 1.5 万，那就给 1.5 万 | 按最低标准给 |
| "我对薪资没要求" | 要么能力不行，要么不自信 | 被压价 |
| "跟市场价一样就行" | 给市场价里最低的 | 自降身价 |
| "要 2 万，因为有房贷" | 越困难越需要工作，可以压价 | 反而被压价 |

**核心原则：不要给区间，要给具体数字。**

### ✅ 推荐话术

**方案一：强调发展 + 行业标准**
```
"我知道应届生/这个岗位薪资有范围，了解到行业内
这个岗位平均水平是 XXX，希望贵公司能按正常体系来定。"
```

**方案二：结合经验/背景**
```
"通过市场调研，我了解到这个岗位薪资水平在 XX-XX，
我的期望薪资是 XX。我有 XX 项目经验，
能够快速上手，相信能为团队创造价值。"
```

## 第三步：薪资超预算时的争取话术

### 情况一：更看重长期发展（柔性争取）

```
"如果薪资有差距，我可以适当调整预期——
毕竟我更看重长期发展，贵公司的平台和这个岗位的成长空间很吸引我。
不过想确认一下，后续薪资调整的机制是怎样的？
比如转正后、以及入职一年后的涨薪幅度大概是多少？"
```

### 情况二：手上有其他 Offer（强势争取）

```
"谢谢认可，但这个薪资确实低于我的预期。
我手里还有其他公司 XX 薪资的 Offer，
不过我对咱们公司更感兴趣（具体说原因：技术栈/团队/发展空间），
所以希望您能再协调一下，我的目标薪资是 XX。"
```

### 情况三：用能力和项目说话

```
"我理解公司有预算考虑，但我认为我的能力配得上 XX 薪资。
在上一家公司/学校期间，我做过 XX 项目（具体成果），
这些经验让我能比一般候选人更快上手。
能否请您向上级再争取一下？"
```

## 谈薪前必做的准备工作

### 1. 市场调研（必须完成）

- **脉脉/牛客网**：查看同岗位薪资水平
- **学长学姐/同行**：打听该公司往年 Offer 情况
- **行业报告**：了解 AI PM 岗位平均薪资区间

### 2. 制定谈薪策略

```
期望薪资  = 市场平均 + 自身溢价（经验/学历/项目）
给出价格  = 期望薪资 × 110%（留出谈判空间）
心理底线  = 期望薪资 × 90%（低于这个就拒绝）
```

**举例（AI PM 社招）：**
- 市场平均：25K
- 有 2 年 AI 产品经验 + 大厂背景：溢价 20% = 30K
- 给出价格：30K × 110% = 33K
- 心理底线：30K × 90% = 27K

## 薪资构成知识（必知）

| 组成部分 | 说明 | 必问的问题 |
|---------|------|-----------|
| **固定工资（底薪）** | 每月固定发放 | 试用期是转正的多少比例？ |
| **绩效** | 根据考核发放 | 占比多少？考核标准是什么？ |
| **奖金** | 年终奖/季度奖 | 有没有？什么时候发？几个月工资？ |
| **补贴** | 住房/交通/餐补 | 具体金额是多少？ |
| **期权/股票** | RSU/ESOP | 归属计划是什么？估值几何？ |

**关键知识点：**
- 谈薪默认说**税前**薪资
- 法律规定试用期薪资不能低于转正的 80%
- 13 薪是固定的，年终奖是浮动的，要问清楚

## 实战对话完整模拟

**场景：AI PM 社招，期望薪资 35K**

**HR：** "你期望的薪资是多少？"

**你：** "面试官您好，因为我对贵公司的薪资结构不太了解，能否先简单介绍一下这个岗位的薪资构成？"

**HR：** "我们这边是固定工资加年终奖，绩效按季度考核。"

**你：** "了解了。基于这个结构，我的期望月薪是 **35K**。我调研了市场上 AI PM 岗位的薪资水平，加上我有两年 AI 产品经验，主导过 RAG 系统从 0 到 1 的落地，上线后客服人工处理量降低 40%，我认为这个薪资是合理的。"

**HR：** "你的背景不错，但我们这个岗位预算在 30K 以内。"

**你：** "我理解公司有预算考虑。我想补充一下，我在前一家公司的项目独立推动了 AI 评估体系从零到有，缩短了模型迭代周期从 4 周到 2 周，这类经验入职后能直接复用。另外我对贵公司在 Agent 方向的布局很感兴趣，是真心想长期发展的。能否向上级再争取一下？或者告诉我转正后的调薪节奏是怎样的？"

## 核心心法

✅ **要做市场调研** — 知道行情才有底气  
✅ **要给具体数字** — 不要说范围，不要说"随便"  
✅ **要有理有据** — 用项目成果和市场行情说话  
✅ **要敢于争取** — 谈薪是双向选择，合理争取很正常  

❌ **不要给薪资范围** — HR 会按最低标准给  
❌ **不要谈个人困难** — 公司要的是价值  
❌ **不要一听超预算就慌** — 合理争取显示应变能力
