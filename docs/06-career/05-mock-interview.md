# AI 模拟面试

> 选择面试类型和时长，AI 面试官会根据你的简历和 JD 进行全程模拟，结束后生成五维诊断报告。

## 使用说明

1. 选择面试类型：技术面 / 产品面 / HR 面 / 压力面 / 综合面
2. 选择面试官风格：专业严谨 / 随和引导 / 压力挑战 / 连环追问
3. 选择时长：15 分钟（8题）/ 30 分钟（15题）/ 45 分钟（20题）
4. 可选：粘贴你的简历和目标 JD，让面试官生成更有针对性的题目
5. 面试结束后查看五维诊断报告，找到薄弱项针对性提升

::: warning 注意事项
模拟面试期间尽量用语音输入，回答时不要查阅资料，才能真实评估自己的水平。
:::

## 五维诊断说明

面试结束后，AI 会从以下五个维度给出评分和改进建议：

| 维度 | 考察内容 | 提升方法 |
|------|---------|---------|
| 🧠 技术理解 | RAG/Agent/大模型原理能否清晰解释 | 刷题库必考题，重点背懂原理 |
| 💡 产品思维 | 需求分析、指标设计、方案取舍是否合理 | 练习框架化思维，多做结构化回答 |
| 🗣️ 表达清晰度 | 回答是否有结构、是否抓住重点 | 先说结论，再展开，用 MECE 原则 |
| 🔍 逻辑深度 | 能否推导出关键判断，而不只是堆砌信息 | 练习"为什么"追问，多做 case study |
| 🌐 前沿认知 | 对最新 AI 动态的了解程度 | 每周阅读前沿速递，关注行业热点 |

## 常见面试失分点

- **回答太长**：没有先给结论，直接展开，面试官不知道你要说什么
- **缺少数据**：只说"效果很好"，没有量化指标
- **技术描述不准确**：说"用了 AI"，但说不清楚是什么技术、为什么选这个
- **不追问确认**：遇到模糊题目直接答，没有先确认面试官的考察意图
