# 职业路径规划

> AI PM 的职业发展路径、岗位分类（B 端 / C 端 / 平台型）、大厂 vs 创业公司选择建议。

职业路径规划内容正在整理中，预计 Phase 3 上线。
:::

## AI PM 岗位分类概览

| 类型 | 代表公司 | 核心要求 | 适合人群 |
|------|---------|---------|---------|
| To C AI 产品 | 字节、百度、腾讯 | 用户增长、体验设计、内容理解 | 有 C 端产品经验 |
| To B AI 产品 | 阿里云、华为云、商汤 | 行业理解、方案交付、客户沟通 | 有 B 端或行业背景 |
| AI 基础平台 | 各大云厂商 | 技术理解深度、开发者生态 | 技术背景较强 |
| AI 创业公司 | 各垂直赛道初创 | 全栈产品能力、快速迭代 | 能接受高风险高回报 |

## 职业发展阶段

```
初级 AI PM（0-2年）→ 中级 AI PM（2-5年）→ 高级 / 产品负责人（5年+）
                                              ↓
                                    产品总监 / CPO / 创业
```
