## 求职全流程时间线

```
Week 1-2   职业定位 + 简历打磨
           → 明确目标岗位方向（B/C/平台型 AI PM）
           → 用简历润色器优化简历
           → 整理 3-5 个项目案例（STAR 结构）

Week 3-4   投递 + 面试准备
           → 批量投递目标公司
           → 用 JD 押题器针对每个 JD 定制题目
           → 用逐字稿生成器准备自我介绍和项目介绍

Week 5-6   面试冲刺
           → 每天 1-2 场模拟面试
           → 及时复盘诊断报告，针对性补强
           → 持续关注前沿速递，准备前沿题答案

Week 7+    拿到 Offer
           → 多 offer 对比方法论
           → 薪资谈判逐字稿
```

## 核心建议

::: tip 最重要的三件事
1. **量化每一个成果**：不说"提升了效率"，要说"将人工审核时间从 2 小时/单降至 15 分钟"
2. **讲清技术理解深度**：不需要会写代码，但要能清楚解释 RAG / Agent / Fine-tuning 的原理和适用场景
3. **准备追问**：每个项目经历至少准备 3 层追问，面试官会沿着你的回答深挖
:::
