## 使用说明

1. 确认 API Key 已配置（见上方）
2. 将简历的某个模块（如工作经历、项目经历）粘贴到输入框
3. 选择润色侧重点（量化数据 / AI 关键词 / 表达清晰度）
4. 点击润色，对比前后差异，挑选你认为好的版本

::: tip 润色建议
每次润色一个模块，不要把整份简历一次性粘进去。分模块润色更容易对比和筛选。
:::

## 简历常见问题

### 量化描述怎么写？

把模糊表达转换为数字：

| ❌ 模糊写法 | ✅ 量化写法 |
|-----------|-----------|
| 负责 AI 客服项目 | 主导 AI 客服系统从 0 到 1 落地，覆盖 3 条产品线，日均处理 1.2 万条对话 |
| 提升了用户满意度 | 通过优化 RAG 召回策略，用户满意度从 72% 提升至 89%（+17pp） |
| 推动了团队协作 | 建立 AI 效果评估双周会机制，缩短模型迭代周期从 4 周→2 周 |

### AI PM 简历必备关键词

技术类：RAG、Agent、Fine-tuning、Prompt Engineering、向量数据库、多模态、Function Calling  
产品类：AI 评估体系、Human-in-the-loop、数据飞轮、冷启动、指标体系  
协作类：模型团队对齐、技术方案评审、LLM 选型

### 常见简历误区

- ❌ 把"使用 ChatGPT 辅助工作"当成 AI PM 经验
- ❌ 只写做了什么，不写为什么做这个决策、结果如何
- ❌ 技术术语堆砌，但说不清楚自己真正的参与深度
