# Agent 是什么

> PM 不需要会写 Agent，但必须能清楚解释它——这是 AI PM 的基本门槛。

## 一句话定义

**AI Agent = 能感知环境 + 自主决策 + 执行行动 + 持续迭代的 AI 系统。**

与只会"问答"的聊天机器人不同，Agent 可以主动分解目标、调用工具、根据结果调整策略。

---

## 传统软件 vs AI Agent

| 对比维度 | 传统软件 | AI Agent |
|----------|----------|----------|
| 执行方式 | 按预设规则逐步执行 | 自主规划步骤，动态调整 |
| 输入处理 | 结构化指令/表单 | 自然语言、图片、文件等非结构输入 |
| 错误处理 | 抛出异常，等待人工干预 | 感知错误，尝试重试或换方案 |
| 知识边界 | 代码写死 | 可调用外部工具/搜索/数据库 |
| 适合场景 | 流程固定、规则明确 | 目标明确但路径不确定 |

## Agent 的四个核心能力

```
┌─────────────────────────────────────────────┐
│                  AI Agent                   │
│                                             │
│  感知 → 记忆 → 规划 → 行动                  │
│                                             │
│  感知：理解用户意图 / 读取环境状态           │
│  记忆：短期上下文 + 长期知识库               │
│  规划：将大目标拆解为可执行步骤              │
│  行动：调用工具 / API / 写文件 / 发消息      │
└─────────────────────────────────────────────┘
```

### 1. 感知（Perception）
理解多模态输入：文字、图片、文件、数据库、API 返回值。PM 关注点：**用户的真实意图能否被准确捕捉**。

### 2. 记忆（Memory）
- **短期记忆**：当前对话上下文（受 Context Window 限制）
- **长期记忆**：向量数据库存储的历史信息、用户偏好
- **工具记忆**：代码执行结果、文件内容

### 3. 规划（Planning）
将复杂目标分解为子任务。主流方式：
- **ReAct**：思考（Reason）→ 行动（Act）→ 观察（Observe）→ 循环
- **CoT**（Chain of Thought）：逐步推理，降低出错率
- **Tree of Thought**：同时探索多条路径，选最优

### 4. 行动（Action）
调用工具（Tool Use）是 Agent 区别于普通 LLM 的关键：
- 搜索引擎、代码执行器、文件读写
- 外部 API（发邮件、查日历、更新数据库）
- 其他 AI 模型（多模态处理）

## PM 必须能回答的问题

**Q：Agent 和 Copilot 有什么区别？**

Copilot 是辅助型：人主导，AI 给建议，人确认后执行。  
Agent 是自主型：人给目标，AI 自主规划并执行，过程中可能不需要人介入。

**Q：什么时候该用 Agent，什么时候不该用？**

| 适合用 Agent | 不适合用 Agent |
|-------------|---------------|
| 任务步骤不确定，需要动态规划 | 步骤固定、规则清晰的流程 |
| 需要调用多个工具/系统 | 单一系统内的简单查询 |
| 允许一定失败率（可重试） | 零失败容忍（金融交易、医疗指令） |
| 用户不想手动操作每一步 | 每步都需人工审核 |

**Q：Agent 最大的产品风险是什么？**

1. **幻觉放大**：多步执行中，每步的小错误叠加后可能酿成大问题
2. **不可控行动**：写文件、发消息等不可逆操作一旦出错代价高
3. **成本失控**：Token 消耗随任务复杂度指数增长
4. **用户信任**：用户不知道 Agent 在"干什么"，黑箱感强

## 主流 Agent 框架一览

| 框架 | 适用场景 | PM 了解要点 |
|------|---------|------------|
| LangChain | 通用 Agent，生态丰富 | 业界标准，但复杂度高 |
| AutoGPT | 自主长期任务 | 演示效果好，生产稳定性差 |
| CrewAI | 多 Agent 协作 | 角色分工明确，适合复杂流程 |
| Dify | 低代码 Agent 搭建 | 国内主流，PM 可直接上手 |
| Coze | 国内开放平台 | 字节系，生态集成好 |
